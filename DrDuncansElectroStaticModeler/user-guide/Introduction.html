Dr Duncan's Eelctrostatic Charge Modeler

This program is an attempt to create an accurate, cross-platform, free, open source, model of the concepts of Electrostatics as learned in a Physics course on Electricity and Magnetism. This software is in a beta stage. Current capabilities include:

  • Accept user entered static charge distribution. Charges are entered in Coulombs and x,y coordinates.
  • Produce a 3d surface plot of the entered charge distribution's electric potential or voltage.
  • Produce a 2d vector field representation of the charge distribuiton's electric field. 
  • Save the 2d vector field as a SVG image. 
  • Save the 3d surface plot in PNG format. 
  • Calculate both the electric potential and electric force vector at any point within the charge distribution.
      Planned additional features:
  • Ability to save the user's entered data.
  • Ability to load charge data from external files such as csv.
  • Ability to export the 3d surface plot in SVG format.
  • Ability to interactively change the charge distribution with a mouse and change the image data in real time.
